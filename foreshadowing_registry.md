## SEED TRACKING FORMAT

When planting new seeds:

```markdown
#### S###: [Name]
| Attribute | Value |
|-----------|-------|
| **Planted** | [Day/Scene] |
| **Description** | [What reader sees] |
| **True Nature** | [What it really means] |
| **Payoff** | [When/how resolved] |
| **Status** | ACTIVE / RESOLVED / DROPPED |
```

---

## RULES FOR SEEDS

### Planting
- Establish naturally in scene
- Don't oversell
- Multiple small mentions > one obvious one
- Reader should be able to miss it first time

### Payoff
- Reference the original seed
- Recontextualize, don't just reveal
- Emotional resonance > plot mechanics
- Reward attentive readers

### Maintenance
- Update status when resolved
- Note if seed direction changes
- Don't drop seeds without reason
- Everything planted should bloom
