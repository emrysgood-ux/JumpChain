# MILLENNIUM PLANNING TOOLKIT
## Reusable Tools for Any Period in the 1000-Year Timeline
## Version 1.0

---

# PURPOSE

This toolkit provides reusable checklists, workflows, and templates for planning ANY segment of the 1000-year timeline, from a single scene to an entire century. It integrates all existing protocols into practical workflows.

---

# SECTION 1: QUICK REFERENCE CHECKLISTS

## 1.1 The Universal 8-Point Pre-Planning Check

Before planning ANY timeline segment, verify:

- [ ] **1. TEMPORAL POSITION:** What year/decade/century? What came before?
- [ ] **2. POWER STATUS:** Lambda phase? Celestial visibility? JumpChain status?
- [ ] **3. CHARACTER AGES:** Tenchi? Yosho? Others present?
- [ ] **4. RELATIONSHIP STATE:** Where are all key relationships?
- [ ] **5. SEED STATUS:** Active seeds? Due for harvest? Need reinforcement?
- [ ] **6. CANON ALIGNMENT:** Pre-canon? Canon? Post-canon?
- [ ] **7. THEME COHERENCE:** How does this serve the millennium arc?
- [ ] **8. PROTOCOL LOAD:** Which protocols apply at this granularity?

## 1.2 The Scene-Level 12-Point Check

For individual scene planning:

- [ ] **1. POV:** Whose perspective? (Sheldon/Tenchi/Yosho/Other)
- [ ] **2. VOICE:** Dialect markers loaded? (see Voice Fidelity Protocol)
- [ ] **3. TONE:** Emotional register appropriate? (see Tone Drift Guard)
- [ ] **4. POWER:** Any manifestations? Visible to whom?
- [ ] **5. SEEDS:** Plant? Reinforce? Harvest? (see Seed Registry)
- [ ] **6. DIALOGUE:** Character-specific patterns? (see Dialogue Texture)
- [ ] **7. SETTING:** Location details consistent?
- [ ] **8. TIME:** Season? Weather? Time of day?
- [ ] **9. FORBIDDEN WORDS:** Checked against current phase?
- [ ] **10. PREVIOUS:** Scene continuity verified?
- [ ] **11. NEXT:** Setup for following scene?
- [ ] **12. THEME:** Contributes to arc/chapter/millennium theme?

## 1.3 The Chapter-Level 6-Point Check

For chapter planning:

- [ ] **1. ARC POSITION:** Where in the five-act structure?
- [ ] **2. POV ROTATION:** Balance across characters?
- [ ] **3. PACING:** Scene density appropriate?
- [ ] **4. SEEDS:** Chapter plants and harvests mapped?
- [ ] **5. EMOTIONAL ARC:** Rise and fall tracked?
- [ ] **6. CLIFFHANGER/CLOSURE:** Ending type determined?

# SECTION 2: PROTOCOL APPLICATION MATRIX

## Which Protocols Apply When?

| Planning Level | Required Protocols | Optional Protocols |
|----------------|-------------------|-------------------|
| **Scene** | Voice Fidelity, Dialogue Texture, Tone Drift Guard | Discomfort Flagging |
| **Chapter** | All scene protocols + Structural Density | Theme-to-Scene |
| **Arc** | All chapter protocols + Truth Collapse | Scene Memory |
| **Year** | All arc protocols + Symbol Lifecycle | - |
| **Decade** | Theme coherence + Seed tracking | - |
| **Century** | Millennium arc alignment | - |

## Protocol Quick Reference

### Voice Fidelity Protocol
**Purpose:** Maintain character voice consistency
**Key Elements:**
- Dialect markers (Sheldon: AAVE roots, scientist register)
- Vocabulary restrictions by character
- Internal vs external voice distinctions
- Evolution across timeline (voice changes over 1000 years)

### Dialogue Texture Protocol
**Purpose:** Character-specific speech patterns
**Key Elements:**
- Speech rhythm and pacing
- Habitual phrases
- Register shifts (formal/informal)
- Character-specific vocabulary

### Tone Drift Guard
**Purpose:** Prevent emotional tone inconsistency
**Key Elements:**
- Scene emotional baseline
- Permitted variance
- Transition requirements
- Recovery protocols

### Theme-to-Scene Enforcement
**Purpose:** Every scene serves larger themes
**Key Elements:**
- Millennium themes: family, found belonging, love across time
- Arc-specific themes
- Symbol deployment
- Thematic resonance check

### Truth Collapse Simulation
**Purpose:** Manage mystery/revelation pacing
**Key Elements:**
- What each character knows when
- Revelation cascade effects
- Secret maintenance
- Payoff timing

### Scene Memory Reinforcement
**Purpose:** Long-form narrative recall
**Key Elements:**
- Callback tracking
- Reference consistency
- Memory trigger planning
- Echo moments

### Symbol Lifecycle Protocol
**Purpose:** Consistent symbol usage
**Key Elements:**
- Symbol introduction
- Reinforcement schedule
- Evolution over time
- Final deployment

### Structural Density Optimizer
**Purpose:** Balance scene types and pacing
**Key Elements:**
- Scene type ratios
- Intensity curves
- Breathing room placement
- Climax positioning

### Discomfort Flagging Protocol
**Purpose:** Handle difficult content appropriately
**Key Elements:**
- Content warnings
- Approach strategies
- Reader consideration
- Narrative necessity verification

# SECTION 7: CONSISTENCY VERIFICATION

## Pre-Writing Verification Checklist

Before writing any content for Year Y:

### Temporal Consistency
- [ ] Previous year's events acknowledged
- [ ] Character ages correct
- [ ] School/life stages correct
- [ ] Season/weather appropriate
- [ ] Historical context accurate (for early years)

### Power Consistency
- [ ] Lambda phase correct for year
- [ ] No abilities used before unlocked
- [ ] Celestial grants appropriately masked/revealed
- [ ] JumpChain status correct

### Relationship Consistency
- [ ] All relationships in correct state
- [ ] No character present before introduced
- [ ] Deaths/departures respected
- [ ] Development trajectory maintained

### Seed Consistency
- [ ] No premature harvests
- [ ] No forgotten plants
- [ ] Reinforcement schedule maintained
- [ ] Registry updated
