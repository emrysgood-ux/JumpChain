
### Power Tier Levels

1. **Peak Human** - Possible in real life, extraordinary
2. **Enhanced** - Above human but physically grounded
3. **Supernatural** - Beyond normal physics, limited
4. **Absolute** - Complete mastery within scope
5. **Omni** - All forms/variations of power type
6. **Transcendent** - Evolves beyond limitations

## RECOMMENDED STORY BIBLE STRUCTURE

Based on research, recommended modular structure:

1. **Character Profiles** (Individual files)
   - Infobox data
   - Extended biography
   - Relationship web
   - Power profile references
   - Arc documentation

2. **Power/Ability Profiles** (Linked from characters)
   - VS Battles style classification
   - Superpower Wiki style applications
   - Technique documentation

3. **Location Profiles** (Geographic hierarchy)
   - Infobox data
   - History
   - Notable features
   - Associated events

4. **Organization Profiles** (Faction documentation)
   - Hierarchy
   - Membership
   - Goals/ideology
   - Resources

5. **Timeline/Events** (Chronological record)
   - Event profiles
   - Cross-references
   - Consequence tracking

6. **World-Building Framework** (Setting bible)
   - Comprehensive world details
   - Rules and systems
   - Cultural documentation

7. **Item/Artifact Profiles** (Significant objects)
   - Properties
   - History
   - Ownership chain

8. **Species/Race Profiles** (Group documentation)
   - Biology
   - Culture
   - History
   - Notable members
