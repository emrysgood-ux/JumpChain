# PART II: SCENE STATUS DEFINITIONS

## 2.1 Status Levels

| Status | Definition |
|--------|------------|
| **FINAL** | Complete, edited, ready for publication |
| **DRAFT** | Written but needs revision |
| **OUTLINE** | Structure defined, not written |
| **PLANNED** | Concept exists, no detail |
| **GAP** | Needed but not addressed |

## 2.2 Quality Tiers

| Tier | Name | Criteria |
|------|------|----------|
| 1 | Reject | Breaks laws, violates POV |
| 2 | Functional | Meets minimum, serves plot |
| 3 | Competent | Solid craft — **MINIMUM** |
| 4 | Excellent | Memorable prose |
| 5 | Exceptional | Award-worthy |
