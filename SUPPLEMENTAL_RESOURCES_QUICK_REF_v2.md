## CORE PHILOSOPHY

> **"Have the skill and not need it instead of need the skill and not have it."**

This is NOT power fantasy. This is intelligent preparedness.

**Character Archetype:**
- **Martial Pacifist:** Skilled fighter who prefers peace
- **Technical Pacifist:** Avoids lethal force when alternatives exist  
- **All-Loving Hero:** Believes in redemption, tries talking first, but Moon Tiara Action exists for a reason

## QUICK DECISION MATRIX

| Question | If YES | If NO |
|----------|--------|-------|
| Does it enhance daily life? | ✅ INCLUDE | Continue |
| Does it support relationships? | ✅ INCLUDE | Continue |
| Does it provide reasonable preparedness? | ✅ INCLUDE | Continue |
| Does it offer magical utility? | ✅ INCLUDE | Continue |
| Is it funny/interesting? | ✅ INCLUDE | Continue |
| Does it make him a cosmic threat? | ❌ EXCLUDE | Continue |
| Does it compel conflict-seeking? | ❌ EXCLUDE | Continue |
| Is it domination/conquest focused? | ❌ EXCLUDE | Continue |
| Is it pure edgelord content? | ❌ EXCLUDE | ✅ INCLUDE (Default) |

**Default Action: INCLUDE**

## INCLUSION CATEGORIES

### A: Daily Life Enhancement (PRIMARY)
- Food production, cooking, preservation
- Crafting, tool-making, household items
- Farming, gardening, growth magic
- Cleaning, organization, temperature control
- Entertainment, music, games

### B: Relationship Building (HIGH)
- Languages, translation, empathy
- Healing (medical, emotional, spiritual)
- Teaching, mentoring, knowledge transfer
- Trust-building, protection auras

### C: Preparedness (VALID)
- Combat capability, martial arts
- Defensive barriers, wards, shields
- Danger sense, threat awareness
- Emergency response, rescue skills

### D: Magical Utility (BROAD)
- Elemental manipulation (all elements)
- Enchantment, item enhancement
- Summoning helpers, materials
- Transformation, material conversion

### E: Funny/Interesting (WELCOME)
- Absurd-but-useful abilities
- Comedic potential situations
- Unique creative applications
- Character bonding moments

---

## EXCLUSION CRITERIA (NARROW)

**ONLY exclude:**

1. **Cosmic Threat Status** - Would the Juraian Empire target him?
2. **Conflict-Seeking Mechanics** - Berserker rage, bloodlust, compulsion to fight
3. **Domination/Conquest Focus** - Mind control for empire-building
4. **Pure Edgelord Content** - No narrative purpose beyond shock

## TEST BATTERY

### The Katsuhito Test
> "Would Yosho have this but rarely use it?"
- YES → **INCLUDE**
- Would make him seek fights → **EXCLUDE**

### The Tenchi Test
> "Can you have this and prefer breakfast to battle?"
- YES → **INCLUDE**
- Makes fighting the default → **EXCLUDE**

### The Sailor Moon Test
> "Would Usagi use this to help friends?"
- YES → **INCLUDE**
- Requires being evil → **EXCLUDE**

### The Daily Life Test
> "Can this appear in a slice-of-life scene?"
- Making breakfast, helping with homework, playing → **INCLUDE relevant perks**
- Combat-only application → **Still include if passes other tests**
